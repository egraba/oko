# oko-client
Agent collecting information about the OS on which it is installed.

## Supported OSes
* macOS (11 & 12)

## Collected data

### Machine data
* Machine info (Serial number, type, model)
* Hostname
* IP address (only IPv4)
* MAC address
* CPU info (architecture, model, number of CPUs)
* Physical memory
* OS info (name, release, version)

### Usage data
* CPU
* Memory
* Swap

# oko-client
Agent collecting information about the OS on which it is installed.

## Supported OSes
* macOS
** Monterey (12)
** Big Sur (11) 

## Collected data

### Machine data
* Hardware info
** Serial number
** Type
** model
* Network info
** Hostname
** IP address (only IPv4)
** MAC address
* CPU info
** Architecture
** Model
** Number of CPUs)
* Memory info
** Physical memory
** Swap
* OS info
** Name
** Release

### Usage data
* CPU
* Memory
* Swap
* Network
